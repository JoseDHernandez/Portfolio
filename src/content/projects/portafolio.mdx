## Proceso de desarrollo

Este proyecto nació como parte de un trabajo para la Universidad. La aplicación pasó por tres fases de desarrollo:

- **En la primera**, se implementó utilizando **HTML**, **Javascript** y **CSS de Bootstrap** (mediante CDN).
- **En la segunda**, se integró **SASS** para dar estilo de forma más organizada.
- **En la última**, el proyecto se migró a **Angular** para dar paso a una estructura más robusta y dinámica.

Debido a que era necesario demostrar la evolución del proyecto, en la aplicación están presentes tanto clases de Bootstrap como estilos en SASS. Esto no es lo más óptimo, ya que Bootstrap también ofrece una variante en SASS, pero así se puede notar el progreso en cada etapa. Por otro lado, el paso de HTML a Angular fue más sencillo de lo que esperaba, puesto que toda la estructura estaba ya organizada en componentes de Javascript (ES6).

Este proyecto me ayudó tanto a aprender **SASS** como a tener un **acercamiento a Angular**, que, aunque actualmente no lo utilizo, me proporciona una base más clara en el manejo de frameworks. Esta holística me hubiera sido util para mi otro proyecto de [Cartas contra la humanidad](/proyectos/cards-against-humanity) con React, puesto que no estaba tan familiarizado con frameworks robustos como Next.js o Angular.

## Estructura del Proyecto

A continuación se describe la estructura de archivos y carpetas de la aplicación:

| Archivo      | Descripción                                                                |
| ------------ | -------------------------------------------------------------------------- |
| `index.html` | Plantilla de la pantalla principal                                         |
| `movie.html` | Plantilla de la pantalla de una película, utiliza el `id` pasado en la URL |
| `404.html`   | Página mostrada cuando el `id` de la película no existe                    |

| Carpeta            | Descripción                                                              |
| ------------------ | ------------------------------------------------------------------------ |
| `css/`             | Almacena el CSS resultante de compilar SASS                              |
| `images/`          | Contiene las imágenes de las películas                                   |
| `js/`              | Incluye los scripts de la aplicación                                     |
| `js/components/`   | Contiene los componentes usados en varias páginas, como cartas o reseñas |
| `scss/`            | Archivos fuente en SASS                                                  |
| `scss/components/` | Styles específicos de cada componente                                    |

| Componente         | Descripción                                                                      |
| ------------------ | -------------------------------------------------------------------------------- |
| `formReview.js`    | Formulario para dejar reseñas; valida y almacena en `sessionStorage`             |
| `movieCard.js`     | Muestra la información de una película en forma de tarjeta                       |
| `movieCardMini.js` | Versión más compacta de `movieCard.js`; usada en recomendaciones                 |
| `reviewHTML.js`    | Genera la plantilla HTML de cada reseña                                          |
| `score.js`         | Convierte una calificación numérica en iconos de estrellas                       |
| `theme.js`         | Implementa el cambio de tema (claro/oscuro) y lo deja guardado en `localStorage` |
